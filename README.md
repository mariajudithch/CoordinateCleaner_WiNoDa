This repository contains all material for the WiNoDa School 2025 workshop on spatial data quality and coordinate cleaning using the R package **CoordinateCleaner**.  
The tutorial demonstrates how to download, inspect, and clean herbarium occurrence coordinates, with a focus on recurrent issues such as:

- Invalid coordinates  
- Zero coordinates (0,0)  
- Sea coordinates for terrestrial taxa  
- Country mismatches  
- Capital/Centroid coordinates  
- Institution-based coordinates  
- Duplicate collection records  

The workflow is fully reproducible and designed for participants with diverse academic backgrounds, including biology, physics, anthropology, chemistry, and data science.
